# 如何使用

### 1. 使用前准备

因为大部分服务都是由 OpenAI 驱动的，因此在使用之前，您必须配置 API 密钥，否则无法正常使用。

<Image
  className="w-[1920px]"
  src="/client/guild-1.png"
  alt="guild-1"
  width={1920}
  height={1055}
/>

### 2. 配置 API 密钥

您必须先配置好密钥之后，才能正常使用应用。

默认情况下，使用的是 OpenAI 官方的服务，如果您有自己的代理，可以在配置中填写。

如果你不方便使用 OpenAI 官方服务，也可以尝试下 [**Le-AI API**](/server/introduction) 进行使用。

选择合适的模型后，在输入框开始会话即可。
