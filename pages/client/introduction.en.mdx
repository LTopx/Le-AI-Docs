## What is Le-AI

Le-AI is an open-source third-party client based on the **OpenAI API**. It is developed using **Next.js** and packaged as a client with the help of **Tauri**.

## Clients

The support for the mobile app is still planning.

- [x] Web
- [x] macOS
- [x] Windows
- [x] Linux
- [ ] iOS
- [ ] Android
